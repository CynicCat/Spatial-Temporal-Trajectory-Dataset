# Target

## Description

Target-domain check-in data for cross-city transfer learning. Contains check-in records from cities used as the target (adaptation) domain. Includes both real check-in data (from Foursquare, Brightkite, AgentMove) and synthetic data generated by GPT-4o.

## Data Format

| File | Records | Description |
|------|---------|-------------|
| `target_NewYork2.csv` | 380,248 | New York check-ins (Foursquare format) |
| `target_agentmove_London.csv` | 188,531 | London check-ins (AgentMove format) |
| `target_agentmove_NewYork.csv` | 380,248 | New York check-ins (AgentMove format) |
| `target_brightkite.csv` | 4,747,288 | Brightkite global check-ins (with city labels) |
| `target_bk_gpt4o.csv` | 4,747,288 | GPT-4o generated check-ins (Brightkite format) |
| `target_foursquare.csv` | 33,263,634 | Foursquare global check-ins (with city labels) |

## Data Schema

### Foursquare-derived format

```
City,User ID,Time Offset,Venue ID,utc_time,Longitude,Latitude,Venue Category Name
```

### AgentMove-derived format

```
City,User ID,Time Offset,Venue ID,UTC Time,Longitude,Latitude,Venue Category Name
```

### Brightkite format

```
City,User ID,Time Offset,Venue ID,UTC Time,Longitude,Latitude,Venue Category Name
```

| Field | Type | Description |
|-------|------|-------------|
| `City` | string | City/location name |
| `User ID` | int | Anonymous user ID |
| `Time Offset` | int | Timezone offset in minutes |
| `Venue ID` | string | Venue identifier |
| `UTC Time` | string | UTC timestamp |
| `Longitude` | float | Longitude |
| `Latitude` | float | Latitude |
| `Venue Category Name` | string | POI category |

## Example

```
City,User ID,Time Offset,Venue ID,UTC Time,Longitude,Latitude,Venue Category Name

```

## Task

Target domain for cross-city transfer learning — evaluate how well a mobility model adapts from source cities to new target cities.

## Source

Multiple LBSN sources (Foursquare, Brightkite, AgentMove) + GPT-4o synthetic data, via [AI4DataSynth](https://github.com/AI4DataSynth/Spatial-Temporal-Trajectory-Dataset).
